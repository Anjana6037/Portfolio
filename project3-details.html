Project Details

BloodBank: Donor Information App

BloodBank is a Flutter-based mobile app that helps users store and manage blood donor information in a simple and efficient way. The project includes a full user flow with a splash screen, secure login, and signup pages with proper validation. After authentication, users can add donor details—name, phone number, and blood group—through a floating action button that opens an input dialog. All information is saved locally using SharedPreferences and displayed as clean, organized cards with edit and delete options. With its red and white theme representing blood donation awareness, this app serves as an effective tool for managing donor data and is ideal as a Flutter mini project.

Key Features and Workflow
1. User Authentication
Secure Signup with username and password
Validation to prevent empty fields or duplicate usernames
Login screen checks if username and password match stored data
Stores user credentials using SharedPreferences
2. Home Page Dashboard
Displays all saved donor records in a clean layout
Consistent Red & White theme for blood donation awareness
User-friendly UI with an AppBar and well-structured design
3. Add Donor Information
Add donor Name, Phone Number, and Blood Group
Uses a Floating Action Button (FAB) to open an input dialog
Proper validation ensures correct data entry
Data is stored locally for offline use
4. Manage Donor Records
Donor information appears as cards on the Home Page
5. Smooth User Experience
Full offline functionality
Clean animations and fast navigation
Simple and intuitive interface suitable for all users
